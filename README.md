# ✉️ AI Email Generator using LangChain

## 💡 Overview
This project uses a Large Language Model (LLM) to generate professional email replies based on user input.

## 🚀 Features
- Role-based responses (HR, Manager, Team Lead)
- Tone customization (formal, friendly, polite)
- Dynamic prompt engineering

## 🛠️ Tech Stack
- Python
- LangChain
- OpenAI API

## ⚙️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Add your OpenAI API key in app.py

3. Run:
   python app.py

## 📌 Example
Input:
- Role: HR
- Tone: Formal
- Context: Candidate asking for interview update

Output:
- Professional email reply generated instantly
